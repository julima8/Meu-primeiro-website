Primeiros Passos com HTML

Para iniciar na área de programação é importante que se tenha conhecimentos sobre o que é e como funciona o HTML. HTML é a linguagem padrão para criação de páginas web. Utiliza marcação para estruturar e apresentar conteúdo, como texto, imagens e links, sendo interpretado pelos navegadores para exibição ao usuário.

1. História da Web

A Internet tem suas raízes em uma rede experimental chamada ARPANET, uma agência do Departamento de Defesa dos Estados Unidos. Ela foi projetada para permitir a comunicação entre computadores de pesquisa em diferentes locais, inicialmente para compartilhamento de recursos e troca de informações entre instituições acadêmicas e militares.

Ao longo das décadas de 1970 e 1980, a ARPANET evoluiu e se expandiu para incluir outras redes de pesquisa e educação, formando o embrião da Internet. Em 1983, a ARPANET adotou o protocolo TCP/IP, que se tornou o padrão para a comunicação de dados na Internet.

A partir daí, a Internet começou a crescer rapidamente, com a entrada de universidades, instituições de pesquisa, empresas e, eventualmente, do público em geral. O desenvolvimento de tecnologias como a World Wide Web, criada por Tim Berners-Lee em 1989, e o navegador Mosaic, lançado em 1993, tornou a Internet mais acessível e fácil de usar, levando a uma explosão no uso da rede e à sua popularização em todo o mundo.

2. O que são Clients?

Na web, um Client (ou browser) é um software que permite aos usuários acessar e visualizar informações disponíveis na internet. Ele interpreta os dados recebidos dos servidores web e os exibe de forma legível para o usuário. Os clientes web mais comuns incluem navegadores como Google Chrome, Mozilla Firefox, Safari e Microsoft Edge. Esses programas permitem que os usuários acessem páginas da web, interajam com conteúdo online e utilizem serviços disponíveis na internet.

Aplicações Web

Aplicações Web são soluções criadas que possuem a internet como meio de comunicação entre Cliente e servidores, não sendo necessário sua instalação.

São consideradas aplicações Web quando utilizamos determinado programa através de um navegador, por exemplo: www.youtube.com. Não são consideradas aplicações web quando o programa é utilizado através de um aplicativo instalado no seu dispositivo.

3. O que são Serves?

Servers ou servidores web são computadores ou sistemas de software responsáveis por armazenar, processar e disponibilizar páginas da web para os clientes (navegadores) que as solicitam. Eles são essenciais para a disponibilização de sites e aplicações web na internet.

Os servidores web recebem as solicitações dos clientes, interpretam essas solicitações, processam os dados necessários e enviam as respostas de volta aos clientes, permitindo que os usuários visualizem o conteúdo desejado em seus navegadores. Exemplos de servidores web populares incluem o Apache HTTP Server, o Nginx e o Microsoft Internet Information Services (IIS).

Entendendo a publicação na internet

Para publicar um site na internet, você geralmente segue estes primeiros 3 passos:

  1. Desenvolvimento do site: Crie seu site usando linguagens de marcação como HTML, estilize-o com CSS e adicione interatividade com JavaScript. Você também pode usar frameworks ou sistemas de gerenciamento de conteúdo (CMS) como WordPress, Joomla ou Drupal.
  2. Registro de domínio: Escolha e registre um nome de domínio único (exemplo: www.seusite.com) por meio de registradores de domínio.
  3. Hospedagem: Escolha um provedor de hospedagem web para armazenar seus arquivos e disponibilizar seu site na internet.

Existem outros passos que precisam serem feitos para uma publicação na internet, porém em se tratando de HTML, esse são os principais tópicos que precisamos saber e pesquisar mais a fundo.

4. Linguagem de Programação

Uma linguagem de programação é um conjunto de regras e símbolos usados para escrever instruções que serão executadas por um computador. Essas instruções são escritas de forma lógica e estruturada para realizar tarefas específicas, como manipular dados, realizar cálculos, interagir com o usuário, entre outros.

As linguagens de programação podem ser de baixo nível, mais próximas da linguagem de máquina, ou de alto nível, mais próximas da linguagem humana. Cada linguagem de programação tem suas próprias regras sintáticas e semânticas, e é usada para desenvolver diferentes tipos de software. Exemplos de linguagens de programação populares incluem Javascript, Java, Python, PHP, C#, entre outras.

Linguagem Server-side e Client-side

Uma linguagem server-side é uma linguagem de programação projetada para ser executada no servidor web, em oposição às linguagens client-side, que são executadas no navegador do cliente. As linguagens server-side são usadas para processar solicitações do cliente, acessar bancos de dados, gerar conteúdo dinâmico e retornar respostas para o navegador do cliente. Essas linguagens são essenciais para o desenvolvimento de aplicações web dinâmicas e interativas.
